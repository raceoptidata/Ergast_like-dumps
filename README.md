The RaceOptiData pipeline is a fully automated Python-based ETL that:

- Fetches F1 race data from the Jolpica (Ergast-compatible) API
- Stores it in a normalized MySQL database

- Produces CSV files, a detailed manifest, and documentation
- Uses SQLAlchemy, pandas, Azure Data Factory, and more
